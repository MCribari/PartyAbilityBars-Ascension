# PAB for Ascension Bronzebeard/Malfurion.

**->** Adds a smart party CD tracker that you can anchor to your raid frames.<br>
**->** /pab to open settings.

### Features:
- Track all cooldowns of your teammates
- Support for PvP Trinket. Also handling racials such as Every Man For Himself, Stoneform and WotF
- Fully customizable, you can select which abilities to display and the position of them.
